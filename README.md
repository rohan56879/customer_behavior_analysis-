# customer_behavior_analysis-
End-to-end Data Analytics project focused on analyzing customer purchasing behavior using Python, Pandas, MySQL, and Power BI. The project involves data cleaning, SQL-based data analysis, and interactive dashboard development to generate business insights.

Technical Skills & Tools

Programming: Python
Libraries: Pandas, NumPy
Database: MySQL
SQL Concepts: CTE (Common Table Expressions), Subqueries, Aggregation Functions, Window Functions (ROW_NUMBER), Joins, Group By
Data Visualization: Power BI
Data Processing: Data Cleaning, Data Transformation, Data Modeling


Project Workflow


1️) Data Cleaning & Preparation (Python - Pandas)

Handled missing values and duplicates
Performed data type conversion
Created derived columns for analysis
Prepared structured dataset for SQL processing

2️ SQL Data Analysis (MySQL)

Customer Segmentation (New, Returning, Loyal Customers)
Revenue Analysis by Age Group
Subscription Behavior Analysis
Top Products by Category
Percentage Revenue Contribution
Applied CTEs and Window Functions for advanced querying

3️ Dashboard Development (Power BI)

KPI Metrics: Average Purchase Amount, Average Rating, Total Customers
Revenue Trends Visualization
Subscription Distribution Analysis
Interactive and Business-Oriented Reporting
